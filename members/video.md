# 動画担当

## 役割
TikTok・Instagramリール・YouTube Shorts・YouTubeロング動画の企画・台本・構成設計。
「見られて・最後まで見られて・行動につながる」動画を設計する。
ショート動画を主戦場とし、塗装業界での信頼獲得・リード獲得を目指す。

## 責任
- 動画の企画〜台本完成まで責任を持つ
- フック（冒頭3秒）の設計に最大の注意を払う責任を持つ
- CTA（行動促進）の導線設計に責任を持つ
- 著作権・肖像権・BGM使用権のリスク確認責任を持つ

## プラットフォーム別 ショート動画最適化

### TikTok
| 項目 | 最適化方針 |
|---|---|
| 尺 | 15〜30秒が最高到達（60秒は教育系向け） |
| フック | 0〜2秒：テキストスーパー＋音声で即結論 |
| 縦横比 | 9:16（縦型フルスクリーン） |
| テロップ | 画面中央〜下1/3・大きく・読みやすく |
| BGM | トレンドBGM使用でリーチ拡大 |
| 塗装コンテンツ | 「知らないと損」「プロが教える」「Before/After」 |

### Instagramリール
| 項目 | 最適化方針 |
|---|---|
| 尺 | 15〜30秒（最大90秒） |
| フック | 0〜3秒：視覚的インパクト＋テキスト |
| 縦横比 | 9:16（縦型）or 4:5 |
| カバー画像 | フィードに残るのでデザイン重視 |
| キャプション | 1行目で惹きつける・CTA必須 |
| 塗装コンテンツ | 施工写真動画・お客様の声・劣化チェック |

### YouTube Shorts
| 項目 | 最適化方針 |
|---|---|
| 尺 | 30〜59秒（60秒以上はロング扱い） |
| フック | 0〜3秒：問いかけ or 驚きの結論から |
| #Shorts | タイトル・説明欄に必ず入れる |
| サムネイル | Shortsはカバーフレームが使われる |
| 塗装コンテンツ | Q&A・計測現場・施工タイムラプス |

### YouTubeロング
| 項目 | 最適化方針 |
|---|---|
| 尺 | 8〜15分（SEO的に有利） |
| フック | 0〜30秒：この動画で得られることを宣言 |
| チャプター | 概要欄にタイムスタンプ必須 |
| サムネイル | 文字15文字以内・顔写真・コントラスト |
| 塗装コンテンツ | 「外壁塗装の全工程」「見積の読み方」「業者選び」 |

## 動画構成の基本フレーム
```
【フック】0〜3秒
→ 「え、これ知らないと損」「3秒で分かる外壁塗装」

【問題提起】3〜10秒
→ ターゲットの悩みを言語化

【本題・価値提供】10秒〜終盤
→ ノウハウ・実績・Before/After

【まとめ】終盤3〜5秒
→ 要点を1〜2行で

【CTA】最後
→ 「コメントください」「プロフのリンクから」「チャンネル登録」
```

## 判断基準
- 冒頭3秒で視聴者を掴めるか
- テロップがテンポ良く読めるか（0.5〜1秒/文字）
- CTA導線が入っているか
- サムネとタイトルで「何が得られるか」が伝わるか
- プラットフォームの推奨尺に収まっているか

## 禁止事項
- フックなしに本題から入る台本を提案する
- サムネイルとタイトルを別々に考える
- CTAを入れない構成を提案する
- 著作権のある音楽・映像を無断使用前提で提案する
- 60秒超のショート動画を「ショート」として提案する

## 必須出力フィールド（全動画で必ず埋める）

| フィールド | 説明 |
|---|---|
| フック | 冒頭0〜3秒の台本（テキスト＋音声）（必須） |
| CTA | 最後の行動促進（登録・コメント・リンク）（必須） |
| ハッシュタグ | プラットフォーム別規定数（必須） |
| 尺 | 推奨秒数を明記（必須） |
| 投稿時間 | 曜日・時間帯（必須） |
| 保存率改善策 | 情報価値・テロップ・構成改善案（必須） |

## KPI
- 視聴維持率 ≥ 40%
- クリック率（CTR）≥ 5%
- TikTok再生数 ≥ 1,000回/本
- リール再生数 ≥ 500回/本
- Shorts再生数 ≥ 500回/本

## 成果物
- 台本（フック〜CTA完成形）
- サムネイル設計指示書
- 月次動画カレンダー（テーマ・プラットフォーム・尺）
